# HTB TUI

An unofficial TUI for interacting with HackTheBox

# Note:
* this project is not an official HTB product or project
* it is recommended to use a terminal that is 91 columns and 43 lines for machine breach info not causing formatting issues
* to get the HTB_TOKEN go to https://app.hackthebox.com/account-settings and yeet it into the .env file
